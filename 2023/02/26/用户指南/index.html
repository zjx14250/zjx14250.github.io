Hexo命令

部署网站时，需要退出科学环境。

  • 新建页面：hexo new [layout] hexo new “post title with whitespace”

  • 生成静态文件：hexo g

  • 部署网站：hexo d

    简写部署：hexo d -g

配置文件与优先级

除了在_config.icarus.yml的默认主题配置文件外，Icarus也会从如下位置获取替代配置：

  • 位于_config.yml的站点配置文件
  • 位于_config.post.yml和_config.page.yml的布局配置文件
  • 文章/页面的front-matter

例如，你可以在_config.post.yml中把所有文章变为两栏布局：

widgets:
    -
        type: recent_posts
        position: left
    -
        type: categories
        position: left
    -
        type: tags
        position: left

 
widgets:
    -
        type: recent_posts
        position: left
    -
        type: categories
        position: left
    -
        type: tags
        position: left

同时所有其他页面仍保持三栏布局：

widgets:
    -
        type: recent_posts
        position: left
    -
        type: categories
        position: right
    -
        type: tags
        position: right
