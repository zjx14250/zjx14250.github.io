Markdown排版规范

一、标题格式建议

标题

  1. 文章的顶层标题使用 二级标题，序号用大写数字
  2. 每个小节的标题使用 三级标题
  3. 小节中进一步分层组织时使用 四级标题
  4. 尽量少用 五级标题 和 六级标题，考虑用 有序列表 ctrl+shift+[ 和 无序列表 ctrl+shift+] 代替
  5. 完全不用 一级标题

二、Markdown 标记的典型使用

粗体、斜体、高亮

  • 需要强调某处内容时使用 粗体 ** 内容 ** 或者 ctrl + B

  • 在中文排版中不使用 斜体 ctrl + I。

  • 在英文排版中可用斜体表达强调，或表示书名、题目。

  • 建议在 加粗、斜体、高亮文本 前后加空格

引用

以下情况使用引用标记：

  • 引述内容时（也可直接用引号替代）ctrl+shift+Q 或者 > + 空格

  • 作示例时

  • 给出提示、警告等额外说明时

行内代码

某一行文字中嵌入简短代码时使用 行内代码，如：

转义符号

如果不想让 Markdown 标记生效，可以在标记的每个符号前加上反斜杠 **，这样这些符号将按原样显示，不再具有 Markdown 中的特殊意义。

三、空白符号的运用

空格

  1. 中英文之间，英文前后各加一个空格

  2. 中文和数字之间，数字前后各加一个空格

  3. 若英文或阿拉伯数字若紧邻中文全角标点，则其与标点之间不加空格，如：

    有了 Markdown，可以不再用 Word。

  4. 行内代码的两端各添加一个空格。若行内代码紧邻标点符号，则其与标点之间不加空格

    打开 Linux 虚拟终端，输入 echo 'Hello World'。

  5. 中文与链接之间各加一个空格

空行

正文段落之间用一个空行来分隔，可以显得段落分明、结构不拥挤。Typora 中自动空行了。如：

Markdown 是一种标记语言。在写作时，你的所有文字都是没有样式的纯文本，在其中插入若干 Markdown 标记后，被标记的文字便有了样式。

比如，在你所写的文字中，希望某一行的最终排版呈现一级标题的样式，那就给这行文字加个一级标题的标记；某个地方有两个字需要加粗，那就给这两个字加个粗体标记。
